# Airbnk BLE

Local Bluetooth control for supported Airbnk locks in Home Assistant.

- Requires a working Home Assistant Bluetooth adapter or ESPHome Bluetooth proxy
- Supports BLE auto-discovery when Home Assistant can see a connectable Airbnk lock
- Local runtime operation over Home Assistant Bluetooth
- Cloud-assisted bootstrap acquisition during setup or refresh
- Manual bootstrap path for advanced or recovery use
- HACS custom integration packaging
